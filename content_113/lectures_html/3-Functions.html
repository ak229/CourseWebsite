Functions At Our Disposal:

  • There are tons of functions already defined for us to use!

Python provides many libraries* of code snippets that programmers can use at their whim.


*A library is just a collection of files that can be referenced in a new program for certain functionality. The standard Python library contains print( ), a function we've used to return information to the user.

The most commonly used functions when starting out in Python are:

Function Usage
print(inputVariable) Return information to the console.
input( ) Obtain information from the console.
int(inputVariable) Convert a string to an integer.

Let's pick apart the print(inputVariable) function...



    print - The name of the function.


    (inputVariable) - The function's argument list.


Why write and use functions?

  • Make reusable code that is specific to our needs
  • Make more versatile and flexible code

Your time is valuable. These two points can save you a lot of time that would be spent typing

Function Definition:

                

                
Our add( ) function was defined in two lines:

                

                
def add(x,y):            #function definition
    return(x + y)        #function body (indented with whitespace)
              

                

                

                
The body of any function is dictated by a certain amount of whitespace indentation from the
                    start of the function definition. The line return(x + y) could have been indented by
                    two spaces. If that were the case, the rest of the function body must also be indented by no
                    more and no less than two spaces.

                



                

                
Function Call:

                    code.
