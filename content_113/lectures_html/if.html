else

			
If the if statement is not true, we can have a backup statement, which is "else".

			
You do not need to add any other condition like you do in the if statement. It just takes all opposite possibilities of the if statement's condition.

            
            

                


                    

                    
                

            


            

			
The line(s) in the else statement must execute if the if statement is false.

			

			
If the if statement is true, the line(s) in else statement will not be executed.



            
            

                


                    

                    
                

            


            

			
        



        

			
elif

			
Using another condition if the first if statement is false.

			
You need a condition just like the if statement, and unlike the else statement.

			
A simplication of "else if". In other programming languages like C++, "else if" is used instead of "elif".



            
            

                


                    

                    
                

            


            

			
			
Our first if statement with condition x<=14 is false. Therefore it goes to elif statement with condition x>15. And this condition was true so "x is greater than 15" is printed.



            
            

                


                    

                    
                

            


            

			
This time our if statement with condition x<= 14 was true, and so the lines in the elif statement weren't executed.

When we are just using two if statements, they are independent, and in our example they both have correct conditions that are x>3 and x>1, so "x" was printed twice from each if statement.

			
However, in our if and elif example with variable "y", firstly our if statement with condition y>3 is true, so our if statement worked and the elif statement wasn't even executed even though its condition y>1 is true.
