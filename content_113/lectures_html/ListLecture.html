List

List Overview

  • Definition:
    A list is data structure that holds an ordered collection of items. You can store a sequence of items in a list. For example, you can imagine you have a shopping list where you have a sequence of items to buy.
  • Syntax:
    A list of items should be enclosed in square brackets(i.e. [ ]) so that Python understands that you are specifying a list. And you put commas in between items whereas you probably have each item on a seperate line in your real world shopping list.
  • Operation:
    In the following lecture, we will introduce some simple operations for list. Such as add, remove, slice and so on.

Slicing Operation

            

                Slicing operation can be used to get a sub-sequence of List. The format is:
                
listVariableName[beginIndex : endIndex]

                
In which beginIndex and endIndex can be blank, if so the blank means beginning or ending. The function just slice the List from beginIndex to endIndex - 1.
