Dictionary

Dictionary Overview

  • Introduction:
    A dictionary is like an address-book where you can find the address or contact details of a person by knowing only his/her name, i.e. we associate keys(name) with values(details). Note that the key must be unique just like you cannot find out the correct information if you have two persons with the exact same name.
  • Syntax:
    {key1:value1, key2:value2, ...}
    The Key-Value pairs are separated by a colon and the pairs are separated themselves by commas and all this is enclosed in a pair of curly braces.
Note that the key-value pairs in a dictionary are not ordered in any manner. If you want a particular order, then you will have to sort them yourself before using it.
