HTML - The Supreme Markup Language

W3Schools Tutorial: http://www.w3schools.com/html/

HTMl is a markup language designed to develop web pages and the world!!

  • HTML stands for Hyper Text Markup Language.
  • HTML documents (web pages) are described by HTML tags.
  • There are reserved tags that represent different kinds of web page content.
    • <p> --- </p> Paragraph: organizes the text it contains into paragraph blocks.
    • <body> --- </body> Body: contains all elements that will be visible within the browser.
    • <html> --- </html> HTML: contains all elements the make up the web page. (Includes styling and scripts)

Basic Web Page File:


Structural Tags:

These are some tags required to develop a basic webpage.

  1. The DOCTYPE tag defines the document to be of the HTML type.
  2. As mentioned above, the HTML tags contain all of the web page elements.
  3. The head tags provide information about the document.
    • Any text inside the head tags does not appear on the web page.
  4. The title tags provide a title for the page (appears in the browser tab).
  5. The body tags contain elements that will be visible on the web page.
  6. The h1 describes a heading.
  7. The p tags contain text that will be represented as a paragraph.

Tags are normally in pairs, like <p> and </p>, where the first tag is the start tag, and the second tag is the end tag. Note that the end tag is identical to the start, but is simply preceeded with a forward slash.

The body tag is most important in terms of relaying content. The body of a web page is what contains all of the text links and images the web designer wishes to relay to visitors.


Content Tags:

These are the tags that are required to relay different kinds of content on a page.

Text
  • Boldface
      • <b> Boldfaced Text </b>
  • Italics
      • <i> Italicized Text </i>
  • Underline
      • <u> Underlined Text </u>
  • Superscrips
      • <sup> Superscript Text </sup>
  • Subscript
      • <sub> Subscript Text </sub>
  • Centering Text
      • <center> Centered Text </center>
  • Line Break
      • <br> - Line Break (single tag)

Images:

Images can be added to web pages using an image tag.

  • Image Tag
      • <IMG SRC="filename">

The image tag is self-closing; It does not have an end tag.

The attribute, 'filename,' can be a path to a local file, or a URL to an online image.

Hyperlinks:

  • Hypertext Link Tag
      • <a href="link"> description of link </a>
  • A link can be a reference to another website, or to another place on your page.
  • To link to another part of your web page...
      • <a href="#name"> description </a>
    • (Will jump to separate 'A' tag on the page)
      <a name="name">

Mailto:

  • Mailto tag
  • <a href="mailto:email-address@webpage.com"> description </a>
  • Will open up the client's mail application and start an email to the address in the tag.

Lists:

  • Unordered: Bulleted list

  • Ordered: Numbered list
