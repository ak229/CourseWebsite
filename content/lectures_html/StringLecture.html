A Brief Overview of Objects and Functions

  • Objects
    • Have certain defined properties and characteristics
    • Store certain information about themselves
    • Have a set of defined functions
    • Variables hold instances of these objects
  • Functions
    • Allow you to access or change information about instances of objects

It is not crucial that you have an in-depth understanding of objects, but a degree of understanding makes functions easier to comprehend.

We will cover more on functions later this week.

Here is a quick non-coding example to make these concepts a bit less abstract:

Let's pretend that there is an object called Dog. All Dogs have the following characteristics:

  • fur color
  • favorite toy
  • number of treats eaten today
This is a specific Dog, called an instance of a Dog: Let's assume that this instance of Dog is referenced by the variable Spot. And that all the functions used below are defined for Dog objects to access their characteristics. 
print(Spot.getFurColor())
print(Spot.getFavoriteToy())
print(Spot.getNumberOfTreats())
Let's pretend that this will print out: 
Brown
Blue Tennis Ball
1
What about functions that can change these characteristics? Let's try this: 
Spot.giveTreat()
print(Spot.getNumberOfTreats())
This should now print: 
        2

    stringVariableName.function()

  • count(string x)
                
    This will return the number of times that the string x occurs in the string
