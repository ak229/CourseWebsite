HTML Attributes

Attributes: additional information about HTML elements.

W3Schools Tutorial on: attributes.

  • HTML elements can have attributes nested within them.
  • An attribute simply provides information about its element.
  • Attributes are always specified in the start tag of a particular element
  • Attributes are declared in name-value pairs.
    • I.E: id="PID"

Different Attributes...


The lang Attribute:


The language of the web page being written can be declared inside the <html> tag.

Declaring a language is a crucial attribute for accessibility applications and search engines to obtain information from a page.

The two first letters of lang's value specify the language. If there is a dialect of that language, that is specified with two more letters.



The title Attribute:


Title is metadata about a certain tag. It is often associated with <p> tags, to give the paragraph a description.



The href Attribute:


Recall the <a> tag, which allowed us to link to am email or different web page; href is the attribute that allows the <a> tag to be linkable.



Size Attributes:


Within the <img> tag, we have the ability to change the size of the image being added to the page.

  • The width attribute represents the width of the image in pixels.
  • The height attribute represents the height of the image in pixels.

The path to the image file itself is also an attribute, the src attribute.



The alt Attribute:

The alt attribute will display its text value if the element it is defined in cannot be displayed on the page.

This is not commonly used, as most modern browsers support almost every HTML5 element. In fact, it is mostly used for older pages that may have antequated code in them.


Attribute Best Practices:

When using attributes, it's best to remember these conventions:

  • All HTML elements can have attributes, but there are a select few that do special things.
  • The HTML title attribute provides information that aids software and search engines.
  • The HTML href attribute allows for referencing other pages on the web or a local filesystem.
  • The HTML width and height attributes allow for sizing images.
  • It is recommended that all attribute values be lower case.
  • Attribute values should be surrounded by quotes.

Table Attributes:


Within the <table> tags, we have several attributes to change the way it displays.

  • The colspan attribute make a cell span multiple columns.
  • The rowspan attribute make a cell span multiple rows.
  • The border attribute specifies the border thickness.
  • The width attribute specifies the width of the table much like it did for images. The width can be absolute, or by percentage.

Care must be taken when using colspan and rowspan to make sure the rows and columns have the correct number of cells.

HTML Entities:

Some characters in HTML are reserved for the language itself. In other words, there are characters that will be rendered by the browser as an HTML element rather than the text you wish to portray on the page.

Examples include:

  • & - Ampersand
  • < - Less Than
  • > - Greater Than
  • “   ”- Quotes

To include these characters in a page's text, one must use HTML entities.
