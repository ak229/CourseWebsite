For Loop Overview

  • Definition:
    The for loop is a statement which iterates over a sequence of object. The loop will go through each item in a sequence.
  • Syntax:
    for variable in List:
    In this for statement, the value of variable will be one element in List by sequence for each loop.

Break Statement

            

                

                    
break

                    

                        
    
                            
  • Just a word break in a line.
    • 
                            
  • The break statement will terminate all the remaining loops in a for statement.
    • 
                            
  • The sentences after this loop will continue executing.
    • 
                        

                    

                

            

            
            

                


                    

                    
                

            

            

            
        


        

            
Continue Statement

            

                

                    
continue

                    

                        
    
                            
  • Just a word continue in a line.
    • 
                            
  • The continue statement will terminate only the current loop in a for statement.
    • 
                            
  • Next loop will continue executing.
    • 
                        

                    

                

            

            
            

                


                    

                    
                

            

            

            
        


        

            
While Loop

            

                
While Loop Overview

                
    
                    
  • 
                        
    
                            
    Definition:
    
                            
    The while loop is allows you to repeatedly execute a block of statements as long as a condition is true.
    
                        
    
                    
    • 
                    
  • 
                        
    
                            
    Syntax:
    
                            
    while boolResult :
    
                            
    The lines in while statement will be repeatedly executed as long as the boolResult gives True. Note that boolResult can be gotten from any calculates or comparisons.
