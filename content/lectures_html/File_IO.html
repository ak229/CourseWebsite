File Input - read contents from a file / File Output - write contents to a file


In this lecture we will explore:

  • Reading contents from a .txt file
  • Writing contents to a .txt file
  • Reading contents from a .csv file

Reading from .txt file


Below is the contents of "CSE113.txt"

Hello!
Welcome to CSE 113!
Python is my favorite programming language.
CSE 113 is one of my favorite courses.
I can't wait to see my final project done!


If we use the code below in Python

with open("CSE113.txt") as file_Input: # opening the .txt file
    readlines = file_Input.readlines()  # getting contents from the file
for line in readlines:
    print(line) # printing each line


Or this one below

file_Input = open("CSE113.txt",'r')
for line in file_Input:
    print(line)


then below (output) will be printed.

Hello!

Welcome to CSE 113!

Python is my favorite programming language.

CSE 113 is one of my favorite courses.

I can't wait to see my final project done!


Note that the "CSE113.txt" file must exist in your directory, or else it will throw an error as you are trying to open a file that does not exist.

We could notice that when reading lines from a .txt file, each line adds one extra new line ('\n').

And of course we can get rid of this.


With the code below,

file_Input = open("CSE113.txt",'r')
for line in file_Input:
    print(line.strip())


We can see the lines (output) without inserting new line for each.

Hello!
Welcome to CSE 113!
Python is my favorite programming language.
CSE 113 is one of my favorite courses.
I can't wait to see my final project done!

Using a list


We do not necessarily have to print out the contents from the .txt file. Let's use things we learned from list lecture.

file_Input = open("CSE113.txt",'r')
line_list = []
for line in file_Input:
    line_list.append(line)
    line_list.append(line.strip())
    break
print(line_list)


In the code we are just adding the first line with & without '\n' into a list.

From the code above, we can see output below by printing the list.

['Hello!\n', 'Hello!']

Writing to .txt file


Now let's learn how to write to .txt file.

If you use the code below,

f = open("Write.txt",'w')
f.write("let's learn how to write to a .txt file." + '\n')
f.write("How's your final project going?" + '\n')
f.close()


"Write.txt" file will look like

let's learn how to write to a .txt file.
How's your final project going?

Note that the "CSE113.txt" file will be created with the contents you wrote even if it did NOT originally exist. It it existed already, the contents will be overridden by the contents you write to it.

Reading from .csv (comma-separated values) file


Below is a csv file of Google's stock price (from hw4) that we are going to parse line by line.

Date,Open,High,Low,Close,Volume,Adj Close
2016-03-09,698.469971,705.679993,694.00,705.23999,1418700,705.23999
2016-03-08,688.590027,703.789978,685.340027,693.969971,2063400,693.969971
2016-03-07,706.900024,708.091003,686.900024,695.159973,2985100,695.159973


We can read this csv file line by line with the code below.

dataFile = open("part_of_hw4.csv", "r")
for line in dataFile.readlines():
    print(line)


And then our output is

Date,Open,High,Low,Close,Volume,Adj Close

2016-03-09,698.469971,705.679993,694.00,705.23999,1418700,705.23999

2016-03-08,688.590027,703.789978,685.340027,693.969971,2063400,693.969971

2016-03-07,706.900024,708.091003,686.900024,695.159973,2985100,695.159973


Since each line is separated by commas in csv file, we can separate each element by comma.

dataFile = open("part_of_hw4.csv", "r")
for line in dataFile.readlines():
    print(line.split(","))


We can get lists containing each element like below.

['Date', 'Open', 'High', 'Low', 'Close', 'Volume', 'Adj Close\n']
['2016-03-09', '698.469971', '705.679993', '694.00', '705.23999', '1418700', '705.23999\n']
['2016-03-08', '688.590027', '703.789978', '685.340027', '693.969971', '2063400', '693.969971\n']
['2016-03-07', '706.900024', '708.091003', '686.900024', '695.159973', '2985100', '695.159973\n']


If we want to print out dates (first column in each line), we can do it in this way:

dataFile = open("part_of_hw4.csv", "r")
for line in dataFile.readlines():
    print(line.split(",")[0])

We just needed to add [0], which refers to the first index of the list.

Eventually our output is 

Date
2016-03-09
2016-03-08
2016-03-07
