The style property

In this lecture we will start adding style to our sites using CSS. There are several methods of adding style, the first of which is through the style property. Like the properties we have already seen, the style property follows the syntax of propertyName="value" though the syntax for the value is a bit different. We'll see that the value of a style tag can have multiple statements within it, ech separated by a semicolon with the syntax "name1:value1; name2:value2;". These can be thought of as sub-properties of the style property and the colon can be read as an equal sign.


In this example we are using 3 of the style properties:

  • color:blue;
    • Set the color of the font to blue in this element.
  • background-color:yellow;
    • Set the background color of this element to yellow.
  • text-align:center;
    • Center the text in this element.





Embedded CSS

Using the style property on individual elements can be very cumbersome, especially when we want to make a change to many elements of the same type. To simplify this we will use CSS to make changes to the entire page with one edit which is similar to defining a variable or a function.


Here we are defining style properties for every element of a particular type. This makes the code much more readable and make changes simpler to implement.

Classes

When we don't want to change every element of a particular property, we will use classes. In fact, it is rare that a top level tag should be given a property in CSS. We will use classes to specify which elements should take which style.

External CSS

Storing all style setting in a separate file.

Just like it's not desirable to use the style tag for individual elements, we usually don't want to use embedded CSS for individual pages. Consider building a website with 50 different pages linked together with a menu and deciding that you want to change the background color of every table. This could involve editing every HTML file and changing the property inside the style tags. In the spirit of reducing our typing, there is an answer for this called External CSS.

We will store all the style setting for an entire site in a CSS file and include the file in the head of each HTML file like this: 

<link rel="stylesheet" href="UBStylin.css">

This will include all the setting in UBStylin.css into the current page. This is the external CSS used for this course site: UBStylin.css
