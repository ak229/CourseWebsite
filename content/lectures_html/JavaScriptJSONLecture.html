JavaScript and JSON

JavaScript Objects

In JavaScript, simple objects like the ones that we will be dealing with can be treated like variables with several values. Each value is formatted in the same way: a name in double quotes, followed by a colon and the value associated with that name. All an object's values are separated by commas and enclosed by curly braces.

You can access an object's values by using the name associated with the value using square brackets or the dot operator, as seen in the example below:

//creating the kitten object
var kitten={furColor:"grey", weight: 5.2, numberOfLegs:4, isCute:True, isFluffy:True};
//accessing the values of the kitten object
//this will return the string "grey"
kitten.furColor
//this will return the number 5.2
kitten["weight"]
//this will return the boolean True
kitten.isCute

//reassigns the value of weight to 5.7
kitten.weight=5.7

JSON

JSON stands for JavaScript Object Notation.

It is a tool that uses notation similar to JavaScript, and can, among other things, allow a developer to convert data from Python to JavaScript, and back again.

Objects

Objects in JSON are similar to objects in JavaScript, however names of values in the object must be enclosed in double quotes as follows: 


               var kitten={"furColor":"grey", "weight": 5.2, "numberOfLegs":4, "isCute":True, "isFluffy":True};

Arrays

Most programming languages use arrays as a basic data structure. Arrays are similar to lists in Python, however with the exception that once an array is created, it cannot grow and shrink in size like a list can. Arrays are created by using square brackets to enclose the information that you intend to contain within your array, as follows:

//creates an array that contains the values 1 through 5, with indicies 0 through 4
var arr=[1,2,3,4,5]

//returns 1
arr[0]

//reassigns arr[1] from 2 to 576
arr[1]=576

Some Fun JSON Functions

json.parse(string)

json.parse() takes in a string and will return a JavaScript object with the values defined in the string 


            //make a string that holds proper JavaScript object formatting
            var stringOfObject='{"TheFirstThing":"String1", "TheSecondThing":500, "TheThirdThing": True}'

            // create the object using json.parse()
            var actualObject=JSON.parse(stringOfObject);

            //this would return "String1"
            actualObject.TheFirstThing

Although this doesn't seem all that useful in this particular case because the string is created right before it is parsed to an object, if you are reading data from a text file, all of your inputs will be strings. This function will allow you to 'create' an object in an external file and the parse it into an actual object in code.

If you are going to use an external file, you will need to read from it using XML, as follows:

Using JSON To Convert Between Python and JavaScript

JSON's real usefulness in web development comes from being able to convert data so it is usable by your backend (written in Python), and the frontend (written HTML and JavaScript).

Converting To And From JSON In JavaScript

We've already seen how JSON.parse() can take a properly formatted string and return a JavaScript object with the attributes defined in the string. If we want to do the opposite of this, create a string from a JavaScript object, we would need to use the JSON.stringify() function.
