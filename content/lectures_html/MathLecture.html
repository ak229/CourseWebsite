Converting an integer to a float (decimal)

        
  • Integer*1.0=Float
    • 
        
    When writing a program, we may not always have control over whether or not a number that we're using is an
            integer
            or a floating point number. For example, what if we need a function that only returns an integer? Does that
            mean
            that
            we're stuck only being able to do integer division? Thankfully, no. We can convert an integer to a float
            simply
            by multiplying
            it by 1.0.
    
        
    
            
            
    
                
    

                    
    
                    
                
    
            
    

            
    
        
    
        
    As var3 proves, multiplying a float by 1.0 still results in a float, so even if you don't know if the
            variable
            that you
            are working with is an integer or a float, multiplying by 1.0 will always make sure that you do have a
            float.

    • sqrt(x)
      • 
            
        
                
      • Returns the square root of x
        • 
                
        A note on import and libraries: This function is called a bit differently than the previous
                    two.
                    Whenever a function is called in code, the compiler
                    needs to be able to find the function definition. All the functions that we've used thus far have
                    all
                    been in the
                    standard, default library. However, sqrt() is in a different library, math, and the compiler won't
                    find
                    it in the
                    default library and it won't know what to do. But if we import the math library and tell the
                    compiler to
                    look in
                    the math library when it reaches sqrt(), it will be able to run the program with no problem
        
                
        For example, imagine that you are a compiler, and you are given the following commands to look up in
                    a
                    dictionary
                    and execute, and you are only handed an English dictionary:
                
        
                
                    sit
            leve
            jump
            walk
        
        
                
        
                
        When you reach 'leve', you won't find it in your English dictionary, and, as a compiler, you'd throw
                    an
                    error and
                    stop. But if you were also handed a French dictionary, and the word 'leve' were marked as a French
                    word,
                    you'd be able
                    to look up the word 'leve' in that dictionary, understand that it means to stand up, execute it and
                    then
                    continue
                    with your list. When we import math in the example below, we're telling the compiler that we're
                    going to
                    use functions
                    that are defined in the math library. Then, when we call the function, we add 'math.' before the
                    function so that
                    the compiler knows to look for the declaration in the math library instead of its default
                    library.
