Project Requirements

You will apply the topics covered in this course in your project to make a full-stack website that includes a AJAX call to a python script that connects to a database. In this way you will create an website where users can interact with a database, and each other.

The theme of your site can be anything you choose and does not need to match the ideas that were pitched in homework 3.
